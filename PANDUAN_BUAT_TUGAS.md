## ❓ Masalah Umum

### 1. "Tidak ada tombol Buat Tugas"
**Penyebab:** Belum ada pertemuan

**Solusi:**
```
1. Klik "Tambah Pertemuan"
2. Isi form pertemuan
3. Simpan
4. Tombol aksi cepat akan muncul
```

### 2. "Tombol tidak bisa diklik"
**Penyebab:** Cache browser

**Solusi:**
```
1. Hard refresh: Ctrl + Shift + R
2. Clear browser cache
3. Restart browser
```

### 3. "Form tidak muncul setelah klik"
**Penyebab:** JavaScript error atau route error

**Solusi:**
```
1. Buka browser console (F12)
2. Cek error merah
3. Screenshot dan kirim error
```

### 4. "Error 404 Not Found"
**Penyebab:** Route tidak dikenali

**Solusi:**
```bash
php artisan route:clear
php artisan cache:clear
```
